# 🧊 Isosurface Visualization using VTK

A C++ project that demonstrates 3D isosurface extraction and rendering using the Visualization Toolkit (VTK).

This project uses the **FlyingEdges3D algorithm** to efficiently generate isosurfaces from volumetric data and displays them with interactive controls.

---

## ✨ Features
- ⚡ Fast isosurface extraction using FlyingEdges3D
- 🎮 Interactive 3D rendering (rotate, zoom, pan)
- 🧠 Built using modern C++ and VTK
- 🪟 Real-time visualization window

---

## 🛠️ Tech Stack
- C++
- VTK (Visualization Toolkit)
- CMake
- Linux (tested on Kali/Parrot OS)
